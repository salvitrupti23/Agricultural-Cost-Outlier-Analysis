# Agricultural-Cost-Outlier-Analysis
This project analyzes agricultural cultivation costs using statistical and time series techniques. Outlier detection methods (IQR, Z-Score, MAD) are applied to identify anomalies, evaluate regional disparities, examine cost-yield relationships, and assess economic stress across crops, states, and years in India.

# Outlier Detection in Cultivation Costs

## Project Overview

This project presents a comprehensive statistical analysis of agricultural cultivation costs to identify anomalies, understand economic stress patterns, and evaluate cost-yield relationships. The study applies outlier detection techniques, descriptive statistics, time series analysis, stationarity testing, and regression analysis to agricultural cost data collected across multiple crops, regions, and years.

## Objectives

* Detect outliers in cultivation cost variables using IQR, Z-Score, and MAD methods.
* Analyze variations in labour, seed, fertilizer, and irrigation costs.
* Compare cost anomalies across different states and regions.
* Study year-wise trends and the impact of external shocks on cultivation costs.
* Examine the relationship between cultivation costs and crop yield.
* Identify economically stressed crops and classify the sources of stress.

## Dataset

The dataset contains agricultural cost of cultivation information collected from multiple crops, states, and years in India.

## Key Findings

* Agricultural cost data exhibits strong variability and non-normal distributions.
* Significant outliers were identified across multiple cost variables.
* Labour costs showed the highest variability among input components.
* Sugarcane, Potato, and Onion were identified as highly stressed crops.
* Cultivation costs have increased over time, indicating rising input expenses.
* Most time-series variables were found to be non-stationary.
* Cost anomalies significantly impact crop yield and productivity.
* Regional disparities exist in cultivation cost structures across states.
